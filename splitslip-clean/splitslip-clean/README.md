# SplitSlip

Itemized bill splitter for recurring shared expenses — built with Expo (React Native).

## Features
- Itemized bills: assign each item to one or more people
- GST split by each person's share or equally
- Who-paid tracking with a settle-up ledger
- Local persistence via AsyncStorage

## Running it
```bash
npm install
npx expo start
```
Scan the QR code with the Expo Go app (iOS/Android) to run it on your phone.
